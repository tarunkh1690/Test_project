To execute Random_number.py first install python in your mac or linux machine.
check random module is installed or not, to check the same run following command
# python
>>> import random
if after hitting you are not getting error output such as "ImportError: No module named random" means module is availble else use "pip install random" to install the module
now to run the script use following command 
# python Random_number.py
alternate option
# chmod +x Random_number.py
#./Random_number.py
